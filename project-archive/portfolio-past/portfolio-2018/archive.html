Archive Of Mirrors, 2017
Website / HTML, CSS & JS

An archival project that delves deep into the internet; a temporary and constantly changing landscape. Built using HTML and CSS, this site exhibits found images of people interacting with mirrrors, exposing how humans are both fascinated and afraid of their own reflection. All images have been hunted from the plentiful worldwide web from sources such as Gumtree, Craigslist, eBay, Facebook & Twitter.
